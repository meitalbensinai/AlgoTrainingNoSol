scipy.special.modfresnelp¶

scipy.special.modfresnelp(x) = <ufunc 'modfresnelp'>¶

Modified Fresnel positive integrals

Returns:

fp

Integral F_+(x): integral(exp(1j*t*t), t=x..inf)

kp

Integral K_+(x): 1/sqrt(pi)*exp(-1j*(x*x+pi/4))*fp
