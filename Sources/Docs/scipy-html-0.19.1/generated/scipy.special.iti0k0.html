scipy.special.iti0k0¶

scipy.special.iti0k0(x) = <ufunc 'iti0k0'>¶

Integrals of modified Bessel functions of order 0

Returns simple integrals from 0 to x of the zeroth order modified Bessel functions i0 and k0.

Returns:ii0, ik0
